toggleMenu v0.1

by soulcutter

A simple menu system which uses jQuery's toggle() or jQuery UI's toggle() methods to open and close the menu. The default behavior is that clicking outside of the menu results in the menu closing, and only one menu can be open at a time.

Documentation

Available Options

toggleMenu may take a hash argument to supply the following options:

opener (recommended):
A jQuery element intended to be clicked on in order to display the menu.
transition (default: 'linear'):
A string, the name of an effect.
duration (default: 'fast'):
A string, 'fast' or 'slow', or a number of milliseconds. Defines how long the transition will take.
onClick (default: function() {}):
A function that is called when the menu is clicked upon. This can be used to give behavior to menu items, though for complex menus it is recommended that you attach click events directly to the menu items themselves.
onClickFilter (default: none):
A jQuery selector string. Only matching elements will trigger the onClick function.
